# UI_RPG_AGr

Learning project for EKA 3D Programming course.
Projekta tika izmantoti dažādi OOP principi, tādi kā:
1. Mantošana (tika izveidoti abstraktie kodi, piemēram, "Character", no kura tika ņemti dažādie abstrāktie mainīgie kā HP, CHaracterName utt).
2. Enkapsulācija (tika izamntoti dažādie [SerializedField] privātie mainīgie ar saviem nosaukumiem un struktiem, piemēram, [SerializedField] private string CharName iedēva man mainīt Enemies un Player vārdus).
3. Visbiežāk tika izmantots Polymorphism (metodē tika ņemts abstraktie mainīgie un override, piemēram, tēlu nāve tika izmantota dažādi tāpēc, ka man vajadzēja uztaisīt pašu playeru nāvi kā "game over", bet pretiņieku - kā jaunu pretinieku).
4. Abstrākcija (abstraktie mainigie, kurie tu var nomainīt citus kodus).

Savā darba es mēģināju uztaisīt 3 dažādus prētiņiekus un 3 dažādus ieroču tipus. Arī uztaisīju prētiņieku bildes un "Game over" screen ar restartu.
